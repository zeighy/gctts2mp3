Tips for best results:
  • Group your text into smaller chunks (e.g., paragraphs or sentences) for easier management and more flexible use of the generated audio.
  • Avoid using special characters. The following characters will be stripped from your input: < > ' " { }
  • The ampersand (&) will be replaced with "and".
  • Avoid using contractions. For example, use "we will" instead of "we'll", and "I will" instead of "I'll". This ensures better pronunciation in the generated speech.
  • Use full words and proper punctuation for the best speech quality.
  • For acronyms and special words:
    • To have each letter read individually, add spaces between them. For example, write "K J A" instead of "KJA".
    • For acronyms pronounced as words, consider spelling them phonetically. For example, write "eighties" instead of "ATIS".
    • Test different spellings to find the best pronunciation for unique names or terms.
